# Mundial 2026 Simulator 🏆

Simulador avanzado para el Mundial FIFA 2026 con ranking FIFA dinámico y cuadro eliminatorio interactivo.

## Características
- **Ranking FIFA Real**: Motor de cálculo Elo oficial implementado.
- **Simulador de Grupos**: Registro de partidos con actualización de tablas en vivo.
- **Cuadro Eliminatorio**: Bracket dinámico desde dieciseisavos (R32) hasta la gran final.
- **Penales**: Soporte para definición por penaltis en fases eliminatorias.
- **Diseño Premium**: Interfaz moderna, responsiva y con animaciones de alta fidelidad.
- **PWA**: Instalable en Android y iPhone para uso sin conexión.

## Instalación en el celular
1. Abre la URL en tu navegador móvil.
2. Selecciona "Añadir a la pantalla de inicio".

## Tecnologías
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- IndexedDB para almacenamiento local persistente.
